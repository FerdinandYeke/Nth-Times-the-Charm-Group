User Stories

Jess Dove's User Stories

As a retired elderly chef, I want to get the best quality ingredients to also give the best meal to my loved ones.

As a retired elderly chef, I want to get the ingredients in a timely manner.

As a retired elderly chef, I want to find more convenient ways to not only make the best meal for my family, but to also spend more time with them.

As a retired elderly chef, I want to use an app that does not require that much technical skill to use.

As a retired elderly chef, I want the app to tell if there are any substitutes that are as closest to the ingredients that I use.

As a retired elderly chef who is still volunteering, I want to have Recipete for cases of having a specific ingredient that is not previously in stock.

As a retired elderly chef who is still volunteering, I want to have knowledge of substitutes of ingredients to a specific ingredient with similar or lower cost and rating that is also highly rated.

As a retired elderly chef who is still volunteering, I want to get ingredients that I know customers will like.

As a retired elderly chef who is still volunteering, I want to know the cost of ingredients in nearby stores and compare the cost.

As a retired elderly chef who is still volunteering, I want to schedule the times of ingredients that might be used for customers in a restaurant.

As a retired elderly chef, I would like to see if there are recipes that have similar ingredients to a recipe I plan to cook with.

As a retired elderly chef, I would like to know the ratings of the recipe by how Recipete has shared.

As a retired elderly chef, I would like to know if the recommended recipe is good with substitutes.

As a retired elderly chef, I would like to have a forum that also talks about the recommended recipes that the app recommends.

As a retired elderly chef, I would like to know if there are ways to enhance the recommended recipe if I were to make it again.

Rachel's User Stories

As a user, I want to be able to select recipes from a personal digital cookbook.

As a user, I want to be able to add and remove recipes from a personal digital cookbook.

As a user, I want to search for recipes in the app that will take me less than 30 minutes to cook.

As a user, I want to receive text notifications indicating the current step her shopper/delivery driver is at e.g., “order received”, “shopping”, “on the way”.

As a user, I want to be able to see a status timeline on the app indicating which step I should be at in the cooking process as a motivation and time management tool e.g., “pick recipe”, “select ingredients”, “prep ingredients”, “cook”.

As a user, I want to be able to select multiple recipes and have their ingredients populate one shopping list.

As a user, I want to be able to sort my recipes by cooking method e.g., “crockpot” or “Insta-pot”.

As a user, I want to be able to add recipes from external websites to my personal digital cookbook.

As a user, I want to be able to schedule my delivery for a future date and time.

As a user, I want the app to be able to set reminders on my phone to remind me to start cooking or meal prepping.

As a user, I want to be able to select the amount of servings I want each recipe to make and have that serving size accurately reflected in the recipe ingredient measurements.

As a user, I want to be able to exclude recipes that include common allergens like milk, eggs, gluten, and tree nuts.

As a user, I want to be able to estimate what a full menu costs.

As a user, I want to be able to manually add recipes not previously stored digitally.

As a user, I want the app to be able to give me a step-by-step checklist for creating a dish.

As a student with ADHD, I want to get the ingredients needed delivered for a certain recipe to avoid a complicated grocery run so I can avoid scheduling conflicts.

As a student with ADHD, I want to be able to follow a recipe without having to fear not having the ingredients so I can make a meal with what I have and save money.
